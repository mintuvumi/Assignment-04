1. Difference between getElementById, getElementsByClassName, querySelector, querySelectorAll
Ans.
a. getElementById() → Selects one element by ID.

b. getElementsByClassName() → Selects multiple elements by class name.

c. querySelector() → Selects the first matching CSS selector.

d. querySelectorAll() → Selects all matching CSS selectors.

2. How do you create and insert a new element into the DOM?
Ans.
a. Use document.createElement() to create an element.

b. Use append() or appendChild() to add it to the DOM.

3. What is Event Bubbling? And how does it work?
Ans.
a. Event bubbling means an event starts from the child element and moves up to its parent elements.

4. What is Event Delegation in JavaScript? Why is it useful?
Ans. Event delegation means adding an event to a parent element to handle child elements.
It is useful because it reduces code and improves performance.

5. What is the difference between preventDefault() and stopPropagation() methods?
Ans.
a. preventDefault() → Stops the default browser action.

b. stopPropagation() → Stops the event from moving to parent elements.
